## Instalação

Instale my-project com npm

```bash
# Clone este repositório
$ git clone <https://github.com/luiszkm/DesignerSystem_IgniteLab.git>

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm run dev

```
